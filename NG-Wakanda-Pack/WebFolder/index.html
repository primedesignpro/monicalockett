NG-Wakanda-Pack

 

Build AngularJS expressive interfaces linked to your REST provided Wakanda Application Model.

This solution pack contain several demo pages to learn how to use angular-wakanda

All this pack and its running examples are available from this Github repository: NG-Wakanda-Pack

1 - Integrate angular-wakanda in your project

Thanks to Bower, Wakanda is very easy to integrate into Angular interfaces either by command line or via your own bowser.json file

angular-wakanda only requires one single JavaScript file to include.

2 - Initialize the Wakanda Service

The Wakanda Service init() method lets you choose to load all the available Model (default) or only specified DataClasses

It works as a JavaScript Promise. Let's load everything.

This Wakanda project provides 3 DataClasses (Country, Company, and Employee), all designed in pure JS in Wakanda.

The Wakanda Model can be designed very easily either graphically or programmatically in pure JavaScript.

3 - Get your Data in One line

angular-wakanda lets you have all your Wakanda Model usable directly from Angular based Web Application interfaces.

4 - Get related data out of the box

No joins, no foreign keys, no map-reduce...

Get access to related data directly through your entity object properties

All you have to do is to tell angular-wakanda when you want to load it locally with this related property $fetch() method

Thanks to the Wakanda alias attributes, this example shows the company manager fullname without even fetching the related Manager
